# victormundi.com

The personal portfolio of [Jeroen Sakkers](https://www.linkedin.com/in/victormundi/), founder and product builder in Amsterdam.

The site presents five selected products:

- [HireData](https://www.hiredata.com/)
- Sol — Rafa & Hans
- [First Names](https://firstnames.victormundi.com/)
- yet.
- Eva Keller

## Local development

Requires Node.js `>=22.13.0`.

```bash
npm install
npm run dev
npm run build
```

The site is built with React, Next.js-compatible APIs, vinext and Cloudflare Workers-compatible output.
